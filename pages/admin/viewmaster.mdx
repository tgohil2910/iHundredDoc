# View Master Details

Upon selecting the View Master Report option from the Menu in admin dashboard, This Page will open up.

![View Points Report](src/assets/admin/viewmaster.png)

Here, Admin can see the details of individual student as well as edit them. To Edit Click on the pen icon available at the left of particular row.

![View Points Report Pen](src/assets/admin/viewmaster-pen.png)

Clicking on pen icon, an edit modal dialog will appear. In that modal dialog user details apart from email address can be updated.

![View Points Report Edit Modal](src/assets/admin/editmodal.png)
