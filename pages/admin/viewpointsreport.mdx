# View Points Report

Upon selecting the View Points Report option from the Menu in admin dashboard, This Page will open up.

![View Points Report](src/assets/admin/viewpointsreport.png)

Then, Select a batch of the students

![Select Batch](src/assets/admin/selectbatch-vpr.png)

Once the batch is selected, the list of the detailed points report of students of the selected batch will be displayed.

![Students Points Report](src/assets/admin/studentspoints.png)

There is an option of export to excel is also available.

![Students Points Report Export](src/assets/admin/studentspoints-export.png)
